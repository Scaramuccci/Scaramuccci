# Featured Projects

## UniversalModLoader

A DevOps-focused project that explores how CI/CD practices can be applied to game modding workflows.

UniversalModLoader provides a pipeline for validating mod structures, packaging builds, generating checksums, and automatically publishing releases using GitHub Actions.

The project demonstrates how traditional software delivery practices can be adapted for modding projects and game tooling.

Key features:

- Mod structure and manifest validation
- Automated packaging into release artifacts
- SHA256 checksum generation
- CI/CD pipeline using GitHub Actions
- Automated GitHub release publishing
- Reproducible builds via Docker

🔗 https://github.com/Scaramuccci/universal-mod-pipeline

## CODToUE5

A hobby project exploring how Call of Duty C2M map data can be imported into Unreal Engine 5 for modding and experimentation.

The tool reads map data and reconstructs the level inside UE5 by generating meshes, materials, and lighting using Python scripts and the Unreal Engine API.

This project focuses on understanding how game level data is structured and how it can be reconstructed inside a modern engine environment.

🔗 https://github.com/Scaramuccci/CODToUE5
